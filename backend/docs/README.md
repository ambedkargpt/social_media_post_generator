This directory is the central place for backend documentation.

- `PHASE2_API.md` - News, Questions, and Profile answer API contracts
- `PHASE2_FIELD_TYPES.md` - field-by-field data types for Phase 2 tables
- `PHASE2_SAMPLE_DOCUMENTS.md` - sample JSON-like documents for `news`, `questions`, `user_profile_answers`
- `PHASE3_POSTS_API.md` - Posts APIs for storage and dashboard fetch
- `PHASE3_POSTS_FIELD_TYPES.md` - field-by-field data types for `posts`
- `PHASE3_POSTS_SAMPLE_DOCUMENTS.md` - sample post documents (including variants)
- `NEWS_MIGRATION_NEWS_ID.md` - news_id strategy and current/legacy migration process
- `PHASE4_HARDENING.md` - balanced security/performance/reliability hardening summary
- `DEPLOYMENT_PLAN.md` - end-to-end production deployment proposal (infra, storage, CI/CD, DR)
